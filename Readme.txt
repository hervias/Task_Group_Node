# Task_Group_Node
Api / Client Node

Este repositorio esta creado en Node js, por lo que a continuacion detallare lo necesario para poder instalar e importar el entorno.

1. Se debe instalar la ultima version de node js.
2. Se debe verificar si se tiene instalado npm, de lo contrario se debera instalar.
3. Se debe de importar el contenido de este repositorio al ide de su preferencia, yo utilize Visual estudio code.
4. Se debe abrir la terminal y dirigirse a la raiz del proyecto, una vez alli, se inicia el app con el comando "npm run dev", esto se debe a que cree un demonio que se encarga del inicio automatico del app, por lo que si se realiza algun cambio en el codigo no es necesario reiniciar el app.
5. Cuando el app este lista, mostrara en la consola que esta corriendo en el puerto 3000.
6. Se adjunta Collection para su consumo.
